SSH Server

To use an SSH connection e.g. to an WLC, do the following:

1. Enter the corresponding IP address in the General settings .

2. Ensure that the SSH server is running. To start it navigate to the folder WebSSH2 an run npm start.

3. Click on a link to the SSH server, e.g. in the left menubar. Enter your credentials for the WLC and wait for the connection to be established.



For information about this feature visit the homepage.


Identification of Client

An client can either be identied by its IP or MAC addresse. To specify which should be used on this webpage, change the General settings .

Per default the name of the IP column is Client IP Address and the Mac column Client Mac Address.
